# 🔥 YOLO-MP:Forest Fire Detection Model

This project implements real-time forest fire detection using a custom YOYLO-MP model. The system is trained on a custom dataset to identify fire in images, videos, and live camera feeds, enabling early detection and prevention.

## 🚀 Features

* Real-time fire detection
* Custom dataset training
* Works on images, videos, and webcam
* Fast and accurate predictions using YOLO-MP
